What this theme is

  • Palette: White (#FFFFFF) + neutral greys, Kite-blue primary (#387ED1), Groww-green for gains (#00B386), Kite-red for losses (#EB5B3C), purple accent (#5367FF) for badges.
  • Typography: Inter for everything UI; IBM Plex Mono for all numbers (P&L, prices, qty) — you instantly see "this is a real trading app".
  • No shadows. No gradients (except subtle chart fill). Hairline 1px borders. Rounded-8px corners. Looks like both Zerodha Kite and Groww at the same time.
  • Color is information only: green = gain, red = loss, blue = primary CTA, neutral grey for everything else.
  • Works for all 6 roles — same visual language; just the menu items + KPIs change per role.
